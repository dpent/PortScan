# PortScan
Nmap clone using Berkelay sockets for Linux/Mac and WinSock for Windows.

## Build steps
- Get cmake
- Create a build directory under the root directory
- Go in the directory
- type cmake ..
- type make
- you now have a compiled program

## For gui
### Linux
- install libxinerama-dev && pkg-config && libxkbcommon-dev && libxcursor-dev && libxi-dev
- make sure you also have libx11-dev libxi-dev libgl1-mesa-dev libxrandr-dev mesa-utils
